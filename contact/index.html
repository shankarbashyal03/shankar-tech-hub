Frequently Asked Questions

How do I use the tools?
Simply click on any tool from the homepage. Each tool opens in a new tab with its own interface. All tools work offline once loaded.
Is my data safe?
Yes! All data is stored locally on your device using browser storage. We do not collect or share any personal information.
Can I use this offline?
Yes! ShankarTech Hub is a PWA (Progressive Web App). Once you visit the site, most tools work without an internet connection.
How do I install the app?
Click the "Install App" button on the homepage, or use your browser's "Add to Home Screen" option to install ShankarTech Hub like a native app.
How can I contact support?
You can email us, send a message through the form above, or reach us via WhatsApp by clicking the floating button on any page.
